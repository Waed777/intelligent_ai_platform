# AI-Driven Big Data Risk & Opportunity Platform

This platform analyzes large-scale financial data, detects anomalies,
predicts potential risks, and provides automated AI-driven recommendations.

## Features
- Big Data Simulation
- Anomaly Detection with Isolation Forest
- Predictive Risk Modeling
- AI Automation & Recommendations
- Interactive Streamlit Dashboard with KPIs & Charts
